Hola, amigos entusiastas de la consola. Si eres como yo y echas de menos aquellos viejos tiempos de Norton Commander, tengo fantásticas noticias para ti. He encontrado un moderno gestor de archivos de consola que te traerá de vuelta todos esos recuerdos nostálgicos, Se llama Midnight Commander, y es una verdadera alternativa a Norton Commander para usuarios de consola.

![](/images/blog/nostalgia-de-norton-commander/hero.png)

Y si! aunque Midnight Commander no es para nada nuevo, se que para muchos puede ser un gran y útil descubrimiento, sobre todo para aquellos que han estado alejado de la consola por mucho tiempo.

En lo particular quedé impresionado cuando probé Midnight Commander por primera vez. Tiene la misma interfaz de dos paneles que tanto me gustaba en Norton Commander, con todas las teclas de función y soporte para atajos de teclado que hacían que la gestión de archivos fuera pan comido. Pero lo que realmente diferencia a Midnight Commander son sus modernas funciones y su capacidad de personalización.

Una de las características clave que me encanta de Midnight Commander es su soporte para múltiples sistemas de archivos. No importa si usas Linux, macOS o Windows, Midnight Commander te tiene cubierto. Y con su editor de texto incorporado, puedes editar archivos fácilmente sin tener que cambiar a otro programa.

Pero lo mejor de Midnight Commander es lo personalizable que es. Puedes instalar plugins y configurar los ajustes para que se adapten a tus necesidades. Y con su soporte para archivos comprimidos, renombrado de archivos por lotes y gestión de permisos de archivos, realmente puedes llevar tu juego de gestión de archivos al siguiente nivel.

Entonces, ¿cómo empezar con Midnight Commander? Muy fácil. Sólo tienes que seguir mi guía paso a paso para instalarlo y ejecutarlo en el sistema operativo de tu elección. Una vez que lo haya instalado y ejecutado, podrá navegar por directorios, copiar y mover archivos y buscar archivos como lo hacía con Norton Commander.

Si está buscando una alternativa a Norton Commander que le traiga de vuelta todos esos recuerdos nostálgicos a la vez que le ofrece características modernas y personalizables, entonces no busque más allá de Midnight Commander. Créeme, ¡no te decepcionará!

### Guía de instalación paso a paso:

**Linux**

-   Abre la terminal en tu sistema Linux.

-   Escribe sudo apt-get update para actualizar las listas de paquetes.

-   Escribe sudo apt-get install mc para instalar Midnight Commander.


Una vez que la instalación haya finalizado, escribe mc en la terminal para iniciar Midnight Commander.

**macOS**

-   Abre la terminal en tu sistema macOS.

-   Escribe brew install mc para instalar Midnight Commander utilizando el gestor de paquetes Homebrew.

-   Una vez que la instalación haya finalizado, escribe mc en la terminal para iniciar Midnight Commander.


**Windows**

-   Descarga la última versión de Midnight Commander desde el sitio web oficial (https://midnight-commander.org/downloads/).

-   Extrae el archivo ZIP descargado en una carpeta en tu sistema Windows.

-   Abre la carpeta extraída y ejecuta el archivo mc.exe para iniciar Midnight Commander.
